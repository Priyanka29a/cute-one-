# 💖 Romantic Love Web App

A sweet, animated, interactive love webpage — built with pure HTML, CSS & JavaScript.

## ▶️ Run it (no install needed)
Just double-click `index.html` — it opens in any browser instantly!

## ▶️ Or run with Python (optional)
```bash
pip install flask
python main.py
```
Then open http://localhost:5000

## ✨ Features
- Floating balloons & animated particles
- Typing animation header messages
- "Reasons Why I Love You" card grid
- "You Make My World Better" section
- **Do you love me?** button — YES triggers celebration 🎉, NO button runs away 😡💖
- Surprise popup love messages
- Click anywhere to spawn floating hearts 💕
- Music toggle button 🎵
- Smooth scroll-reveal animations

## 🎨 Customise
- Edit the `reasons` array in `script.js` to add personal messages
- Edit `surpriseMessages` for custom popup notes
- Swap in real photos by replacing `.photo-card` content in `index.html`
